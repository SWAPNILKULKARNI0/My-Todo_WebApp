# My-Todo_WebApp


This is simple  "My To-do Web Application".

Language- python.

Type- web application.

Module- Streamlit.


Functions:-
          - Create todo list.

         - Add todo item.
         
         - Delete todo item.
         
         - Update todo item.
         
         - Checkbox the completed todos.
         
         
Note:- "You can't add the todo item which alredy exist's in a todo list".
